# Stream Dock M18 Lab

Colección personal de proyectos, plugins, escenas, diseños de botones y pruebas desarrolladas para Stream Dock M18.

## Proyectos incluidos

### MetaTrader Plugin
Plugin y bridge local para controlar MetaTrader 4/5 desde Stream Dock M18.  
Incluye pruebas con Node.js, Named Pipes, comandos BUY, SELL, STATUS, cierre de operaciones y control de lotaje.

### Magic 8 Ball
Plugin tipo bola mágica/oráculo para Stream Dock, con respuestas aleatorias, modo visual personalizado y diseño interactivo.

### Volume Control
Botones para subir, bajar y silenciar volumen del sistema desde Stream Dock.

### Button Designs
Diseños visuales personalizados para botones del Stream Dock, incluyendo estilos luxury diamond, hacker green, neon y escenas temáticas.

## Objetivo

Crear una colección organizada de herramientas para convertir el Stream Dock M18 en una consola personalizada para productividad, trading, multimedia y automatización.

## Tecnologías usadas

- JavaScript
- Node.js
- HTML
- CSS
- Stream Dock SDK
- MetaTrader 4
- MQL4
- Windows Named Pipes
- GitHub
